# Fresh Soda - Online Food Ordering System

A responsive web application for ordering fresh, handcrafted sodas with role-based access for customers, staff, and administrators.

## Features

### Customer Features
- Browse menu with search and filter capabilities
- Add items to cart with quantity selection
- View and manage cart items
- Checkout with multiple payment options
- Order tracking
- Contact form

### Staff Features
- View and manage orders
- Update order status (Pending → Preparing → Completed)
- Inventory management

### Admin Features
- Full order management
- Menu item management (add/remove items)
- User management (add/remove users)
- Business analytics
- Logo customization

## Technology Stack

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Styling**: Custom CSS with CSS Grid and Flexbox
- **Icons**: Font Awesome
- **Fonts**: Google Fonts (Inter, Poppins)
- **Storage**: Local Storage for data persistence
- **Deployment**: Vercel

## Installation & Setup

1. Clone or download the project files
2. No build process required - this is a static web application
3. Deploy to Vercel or any static hosting service

### Deploying to Vercel

1. Create a new repository with your HTML file
2. Connect your GitHub repository to Vercel
3. Vercel will automatically deploy your application
4. Your app will be available at `https://your-project-name.vercel.app`
